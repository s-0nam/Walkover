Feature of SCSS

Just like CSS, Sass also supports the @import directive. The @import directive all ows you to include the content of one file in another. The CSS @import directive has a major drawback due to performance i ssues; it creates an extra HTTP request each time you call it. Howeve r, the Sass @import directive includes the file in the CSS; so no extra HTTP call is required at runtime!
